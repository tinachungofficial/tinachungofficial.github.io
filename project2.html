GroupFindr android application development

Project Description

GroupFindr is an Android application that targets stragglers who struggle to find teammates in group projects/assignments in huge classes. This application allows users to enroll in class sessions and find their potential teammates.

Project Details

  • Create wireframes for activities based on feature specifications
  • POST calls to database to return information
  • Implemented integration of Facebook login
  • Coded Java classes for each activity

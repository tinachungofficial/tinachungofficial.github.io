GroupFindr android application development

Project Description

GroupFindr is an Android application that targets stragglers who struggle to find teammates in group projects/assignments in huge classes. This application allows users to enroll in class sessions and find their potential teammates.

Process

The motivation behind creating this application is that every group member had trouble finding groups because we did not know many students in the class. We decided that this tool would have expedited the process of finding teammates who are compatible. We met to discuss the features we wanted for this application. We wanted to create a simple, usable, application that would work well with Facebook. We decided that the application should not have too many contact features. The objective for users is to enroll in a class session and find their potential teammates. They would in turn use Facebook to contact them.

The technologies used were:

  • Android Studio (design)
  • Java (implementation)
  • Amazon Web Services (server)
  • Google Analytics (testing)
  • Facebook API (Facebook login feature)

My Role/Responsibilities

  • Designed the general flow across different activities in the application
  • Create wireframes for activities based on feature specifications
  • POST calls to database to return information
  • Implemented integration of Facebook login
  • Coded Java classes for each activity
